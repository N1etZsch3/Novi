## 🚀 快速开始

### 新人入门路径
1. 阅读 [项目概览](01-入门指南/项目概览.md) 了解项目整体
2. 查看 [项目结构说明](02-架构设计/项目结构说明.md) 理解代码组织
3. 根据需要阅读具体的功能模块文档
4. 参考 [API接口汇总](04-API文档/API接口汇总.md) 进行接口对接

### 前端开发者
→ 直接查看 [API接口汇总](04-API文档/API接口汇总.md)

### 后端开发者
→ 先看 [项目结构说明](02-架构设计/项目结构说明.md)，再看具体模块文档

### 数据库管理员
→ 查看 [数据库设计文档](02-架构设计/数据库设计文档.md)
