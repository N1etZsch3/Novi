# 1.3 会话管理接口文档

本模块用于管理用户的聊天会话列表及历史消息。

## 1. 获取会话列表 (Get Session List)

获取当前用户的所有历史会话列表，用于侧边栏展示。

- **接口地址**: `/api/v1/sessions`
- **请求方式**: `GET`
- **Header**: `Authorization: <token>`

### 响应结果 (Result<List<ChatSession>>)
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "id": "session_123",
      "userId": 1001,
      "title": "关于Java的学习",
      "createdAt": "2023-10-01T10:00:00",
      "updatedAt": "2023-10-01T10:05:00"
    },
    {
      "id": "session_456",
      "userId": 1001,
      "title": "新会话",
      "createdAt": "2023-09-28T09:00:00",
      "updatedAt": "2023-09-28T09:00:00"
    }
  ]
}
```

---

## 2. 获取会话消息详情 (Get Session Messages)

点击某个会话时，加载该会话的所有历史消息。

- **接口地址**: `/api/v1/sessions/{sessionId}/messages`
- **请求方式**: `GET`
- **Header**: `Authorization: <token>`
- **路径参数**:
    - `sessionId`: 会话ID

### 响应结果 (Result<List<ChatMessage>>)
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "id": 501,
      "sessionId": "session_123",
      "role": "USER",
      "content": "什么是Java?",
      "timestamp": "2023-10-01T10:00:00"
    },
    {
      "id": 502,
      "sessionId": "session_123",
      "role": "ASSISTANT",
      "content": "Java 是一门面向对象的编程语言...",
      "timestamp": "2023-10-01T10:00:05"
    }
  ]
}
```
- **role**: `USER` (用户) 或 `ASSISTANT` (AI)。

---

## 3. 删除会话 (Delete Session)

删除指定会话及其所有关联消息。

- **接口地址**: `/api/v1/sessions/{sessionId}`
- **请求方式**: `DELETE`
- **Header**: `Authorization: <token>`
- **路径参数**:
    - `sessionId`: 会话ID

### 响应结果 (Result<Void>)
```json
{
  "code": 200,
  "message": "success",
  "data": null
}
```
