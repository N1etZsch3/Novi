# 1.4 AI 出题功能接口文档

本模块提供 AI 智能出题功能，支持根据指定科目、题型、难度生成题目。

## 1. 生成题目 (Generate Questions)

根据用户配置生成题目。AI 返回的结果通常为 JSON 格式的字符串，前端需要进行二次解析。

- **接口地址**: `/api/v1/question/generate`
- **请求方式**: `POST`
- **Header**: `Authorization: <token>`
- **Content-Type**: `application/json`

### 请求参数 (QuestionGenerationRequest)
| 字段名 | 类型 | 必填 | 说明 | 示例 |
| :--- | :--- | :--- | :--- | :--- |
| subject | string | 是 | 科目名称 | "湖北专升本英语" |
| questionType | string | 是 | 题型 | "语法填空" |
| difficulty | string | 是 | 难度 (simple/medium/hard) | "medium" |
| quantity | integer | 是 | 题目数量 | 5 |
| theme | string | 否 | 出题主题/范围 | "科技发展" |

### 响应结果 (Result<QuestionGenerationResponse>)
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "recordId": 101,
    "questions": "[{\"question\": \"...\", \"options\": [...], \"answer\": \"A\"}, ...]"
  }
}
```
- **recordId**: 本次出题记录的ID，可用于后续查询历史。
- **questions**: AI 生成的题目内容，为 JSON 字符串格式。
