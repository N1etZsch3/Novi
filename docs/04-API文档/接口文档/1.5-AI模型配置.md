# 1.5 AI 模型配置接口文档

本模块用于管理和切换 AI 模型配置（如 GPT-3.5, GPT-4, 豆包等）。

> **安全提示**: 所有接口仅返回安全的 DTO 数据，不包含 API Key 等敏感信息。

## 1. 获取所有模型配置 (List Models)

获取系统中所有可用的 AI 模型列表及其激活状态。

- **接口地址**: `/api/model/config/list`
- **请求方式**: `GET`
- **Header**: `Authorization: <token>`

### 响应结果 (Result<List<AiModelConfigDto>>)
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "id": 1,
      "modelName": "doubao-seed-1-6-flash-250828",
      "description": "豆包模型 - 快速响应",
      "isActive": true
    },
    {
      "id": 2,
      "modelName": "gpt-3.5-turbo",
      "description": "OpenAI GPT-3.5",
      "isActive": false
    }
  ]
}
```

---

## 2. 获取当前激活模型 (Get Active Model)

获取当前正在使用的 AI 模型配置。

- **接口地址**: `/api/model/config/active`
- **请求方式**: `GET`
- **Header**: `Authorization: <token>`

### 响应结果 (Result<AiModelConfigDto>)
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "modelName": "doubao-seed-1-6-flash-250828",
    "description": "豆包模型 - 快速响应",
    "isActive": true
  }
}
```

---

## 3. 切换模型 (Switch Model)

切换当前激活的 AI 模型。切换后，新的聊天请求将使用新模型。

- **接口地址**: `/api/model/config/switch/{modelName}`
- **请求方式**: `POST`
- **Header**: `Authorization: <token>`
- **路径参数**:
    - `modelName`: 目标模型的名称 (如 `gpt-3.5-turbo`)

### 响应结果 (Result<String>)
```json
{
  "code": 200,
  "message": "success",
  "data": "模型切换成功"
}
```
