# Task
请根据用户提供的主题【{theme}】，编写 {quantity} 道标准的 "Part II 连词成句" 题目。

# Constraints & Rules
1. **难度控制**: {difficulty}。句长控制在 8-15 个单词。
2. **题型结构**:
   - 每道题提供一组被打乱顺序的单词或短语，用斜杠 "/" 分隔。
   - 每个分隔的部分不超过**三个**单词。
   - **关键要求**：打乱后的词组必须能组成**唯一**通顺的句子。
   - 必须包含特定的语法考点（如：定语从句、倒装句、被动语态、强调句、非谓语动词等）。
3. **输出格式**: 必须是严格的 JSON 数组。
